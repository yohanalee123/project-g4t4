about mentor.

Hello, we're the developers of mentor.

The reason we decided to set up mentor. is because we believe that there is a need for different people to learn from each other. Everyone can be a mentor. The idea is also created based on the fact that we are the only group that did not bid together, and that when we do this project we can learn from one another.

We thought as a team that Enterprise Solution Development is a very meaningful module as it allows us to have the expertise on how to build an enterprise solution and how to use microservice in the Service-Oriented Architecture Layers.
